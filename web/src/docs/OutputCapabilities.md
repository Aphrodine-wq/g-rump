 ## Batch Export 
  
 ### Export Multiple Animations at Once 
 ```bash 
 grump export --batch \ 
   --format mp4,gif,lottie,sprite \ 
   --resolution 1080p \ 
   --fps 120 \ 
   --animations hero_*.json \ 
   --output ./builds/ 
 ``` 
  
 **Results:** 
 - `hero_jump.mp4` - Video export 
 - `hero_jump.gif` - Animated GIF
