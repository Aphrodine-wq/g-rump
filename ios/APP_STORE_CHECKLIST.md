# App Store Submission Checklist

## Pre-Submission Checklist

### Code & Configuration
- [ ] Xcode project created and configured
- [ ] Bundle Identifier set: `com.yourname.grump`
- [ ] Team/Developer account selected in Xcode
- [ ] Signing & Capabilities configured
- [ ] Production API endpoint configured
- [ ] Info.plist properly configured
- [ ] App version set to 1.0.0
- [ ] Build number set to 1
- [ ] Minimum iOS version: 16.0
- [ ] All code compiles without errors
- [ ] No deprecated APIs used

### App Store Connect
- [ ] App created in App Store Connect
- [ ] Bundle ID registered
- [ ] App name reserved: "Grump"
- [ ] App information filled out
- [ ] Pricing set (Free or Paid)
- [ ] Availability configured
- [ ] Age rating completed
- [ ] Export compliance answered

### Assets Required
- [ ] App Icon: 1024x1024px PNG
- [ ] Screenshot: 6.7" Display (1290 x 2796)
- [ ] Screenshot: 6.5" Display (1242 x 2688)
- [ ] Screenshot: 5.5" Display (1242 x 2208)
- [ ] App Preview Video (optional)
- [ ] App Description (4000 chars max)
- [ ] Keywords (100 chars max)
- [ ] Support URL
- [ ] Privacy Policy URL

### Testing
- [ ] Tested on physical iPhone device
- [ ] All animations work smoothly
- [ ] Chat functionality works
- [ ] Onboarding flow works
- [ ] Settings work correctly
- [ ] Dark mode works
- [ ] Production API accessible
- [ ] No crashes in crash logs
- [ ] Performance is acceptable
- [ ] Memory usage is reasonable

### Legal & Compliance
- [ ] Privacy Policy created and hosted
- [ ] Terms of Service (if applicable)
- [ ] Export compliance information
- [ ] Content rights confirmed
- [ ] Age rating appropriate

### Build & Archive
- [ ] Clean build (Product → Clean Build Folder)
- [ ] Archive created successfully
- [ ] Archive validated
- [ ] Archive uploaded to App Store Connect
- [ ] Build processing completed in App Store Connect

### Submission
- [ ] Version information filled out
- [ ] "What's New" description written
- [ ] Screenshots uploaded
- [ ] App preview uploaded (if applicable)
- [ ] Description and keywords added
- [ ] Review information provided
- [ ] Demo account provided (if needed)
- [ ] App submitted for review

## Post-Submission

- [ ] Monitor App Store Connect for status updates
- [ ] Respond to any review feedback promptly
- [ ] Prepare marketing materials
- [ ] Set up analytics tracking
- [ ] Plan launch announcement
