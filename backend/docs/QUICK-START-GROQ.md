# Quick Start: Groq Integration

## Your Groq API Key Location

**File:** `backend/.env`

**Add this:**

```env
AI_PROVIDER=groq
# Get your API key from: https://console.groq.com/
GROQ_API_KEY=your_groq_api_key_here
GROQ_MODEL=llama-3.1-70b-versatile
PORT=3000
```

## That's It!

1. Create `backend/.env` with the content above
2. Run: `cd backend && npm start`
